# Safety Guardian Arm

The Safety Guardian Arm provides comprehensive security, PII protection, and content filtering for all system operations.

## Features

- Advanced PII detection (18+ types)
- Content filtering and moderation
- Policy enforcement
- Threat detection
- Compliance validation (GDPR, CCPA)

## Architecture

- **Language**: Python 3.11+
- **Framework**: FastAPI
- **LLM**: GPT-3.5-turbo
- **Port**: 8060
